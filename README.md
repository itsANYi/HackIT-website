# 黑客IT社 網站

一個使用 Flask 框架建立的互動式社團網站，具有 Apple 風格的設計。
點這裡進入網站：https://saihs-hackit.eliaschen.dev/login 

## 功能特色

- 🔐 使用者登入/註冊系統（SQLite 資料庫）
- 🚀 創意的系統安裝流程（社團介紹）
- 👤 完整的使用者資訊管理
- 🎮 即時配對問答遊戲
- 💎 毛玻璃特效設計
- 📱 響應式設計

## 安裝步驟

1. 安裝依賴套件：
\`\`\`bash
pip install -r requirements.txt
\`\`\`

2. 執行應用程式：
\`\`\`bash
python app.py
\`\`\`

3. 在瀏覽器中開啟：
\`\`\`
http://localhost:5000
\`\`\`

## 技術棧

- **後端**: Python Flask, Flask-SocketIO
- **前端**: HTML5, CSS3, JavaScript
- **資料庫**: SQLite
- **即時通訊**: Socket.IO
- **設計風格**: Apple-inspired, 毛玻璃效果

## 設計配色

- 主要色: #a0cbb3（薄荷綠）
- 背景色: #020102（深黑）
- 文字色: #ffffff（白色）

## 功能說明

### 1. 登入/註冊
- 使用者可以註冊新帳號或登入現有帳號
- 密碼使用雜湊加密儲存

### 2. 社團介紹（系統安裝）
- 創意的多步驟安裝流程
- 包含使用者條款、路徑選擇、環境檢查等步驟
- 互動式設計，讓使用者體驗更有趣

### 3. 使用者資訊
- 完整的個人資料管理
- 自動儲存功能
- 包含頭貼、暱稱、興趣等欄位

### 4. 社團活動
- 隨機配對問答遊戲
- 即時通訊功能
- 輪次制遊戲機制

## 注意事項

- 確保已安裝 Python 3.7 或更高版本
- 首次執行時會自動建立資料庫
- 預設在 5000 埠執行
